== Installation ==

Install WordPress Posts Widget plugin as a regular WordPress plugin. Here is different ways to install WordPress Posts Widget plugin:

* [Visual Installation & Usage Guide &raquo;](https://docs.gsplugins.com/wordpress-posts-widget)

= Install by Search Plugin =
- In your Admin, go to menu Plugins > Add New
- Search WordPress Posts Widget
- Click to install
- Activate the plugin
- At Appearance > Widgets you'll found - GS Posts Widget

= Install by Upload Plugin =
- Download the latest version of GS Posts Widget (.zip file)
- In your Admin, go to menu Plugins > Add New
- Select **Upload Plugin**
- Click on **Choose File**
- Select downloaded ***gs-posts-widget-pro.zip*** & click on **Install Now** button
- Activate the plugin
- At Appearance > Widgets youâ€™ll found â€“ GS Posts Widget

= Install Plugin using FTP =
- Upload gs-posts-widget-pro folder inside the /wp-content/plugins/ directory
- Go to WordPress dashboard > Plugins > Activate the GS Posts Grid
- At Appearance > Widgets youâ€™ll found â€“ GS Posts Widget
